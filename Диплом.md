# **Тема проекта**: Исследование особенностей организации процесса ручного тестирования на основе интернет-магазина косметики
## **Цель**: Изучить особенности ручного тестирования   

### **Задачи**:   

1. Изучить литературу, касающуюся темы исследования;    

2. Рассмотреть основные виды и методы ручного тестирования;

3. Ознакомиться с основными принципами составления такой тестовой документации как чек-листы, тест-кейсы, тест-планы, баг-репорты; 

4. Составить план ручного тестирования с использованием техники тест-дизайна   

5. Выполнить ручное тестирование   

6. Разработать предложения по улучшению ручного тестирования 

### **Инструменты**: Postman, Qase.io, TestRail или Jira (решу позднее), DevTools, Git
